# Glosario de Rescatto

Este glosario define los términos usados en toda la plataforma. Consulta esta referencia si encuentras una palabra que no conoces.

---

| Término | Definición |
|---|---|
| **Rescate** | Acción de comprar comida que de otro modo se desperdiciaría. Cada compra en Rescatto es un rescate. |
| **Pedido** | Solicitud de compra que un cliente hace a un negocio dentro de la plataforma. |
| **Racha** | Número de días consecutivos en que un cliente realiza al menos un rescate. Las rachas aumentan los puntos que ganas. |
| **Puntos** | Moneda de recompensa dentro de Rescatto. Se acumulan con cada rescate y se pueden canjear por descuentos. |
| **Multiplicador** | Factor que se aplica sobre los puntos base según la racha activa (1.5×, 2×, 2.5×, 3×). |
| **Embajador** | Nivel especial que alcanza un cliente al referir suficientes amigos a la plataforma. |
| **Flash Deal** | Oferta de tiempo limitado publicada por un negocio, con precio especial y stock reducido. |
| **Hot Deal** | Producto con alta demanda reciente, marcado automáticamente por la plataforma. |
| **Precio dinámico** | Precio que baja automáticamente a medida que se acerca la hora límite del producto. |
| **CO₂ ahorrado** | Estimado del dióxido de carbono que no se emitió gracias al rescate de un alimento. |
| **KDS** | Kitchen Display System. Pantalla de cocina donde el personal ve los pedidos en tiempo real. |
| **Wallet** | Billetera virtual dentro de la app donde se acumulan puntos y saldo. |
| **Driver** | Domiciliario que recoge pedidos en los negocios y los entrega a los clientes. |
| **Venue** | Negocio registrado en Rescatto (restaurante, panadería, supermercado, etc.). |
| **VENUE_OWNER** | Rol del dueño o encargado de un negocio registrado en Rescatto. |
| **KITCHEN_STAFF** | Rol del personal de cocina que gestiona la preparación de pedidos. |
| **ADMIN** | Rol administrativo con acceso a gestión de usuarios y negocios. |
| **SUPER_ADMIN** | Rol con acceso total a la plataforma, incluyendo configuración global. |
| **CUSTOMER** | Rol del usuario final que realiza pedidos y rescates. |
| **Checkout** | Proceso de confirmación y pago de un pedido. |
| **Wompi** | Pasarela de pago que procesa las transacciones dentro de Rescatto. |
| **PWA** | Aplicación web progresiva. Rescatto puede instalarse en tu celular como una app nativa. |
| **Stock** | Cantidad de unidades disponibles de un producto. |
| **availableUntil** | Fecha y hora límite hasta la que un producto puede ser rescatado. |
| **Leaderboard** | Tabla de clasificación que muestra los mejores rescatadores de tu ciudad y a nivel global. |
| **Impacto** | Sección de la app donde puedes ver tu huella ambiental positiva y tus logros. |
| **Categoría** | Clasificación de los productos o negocios (ej. Panadería, Restaurante, Supermercado). |
| **Chat** | Sistema de mensajería dentro de la app para comunicarte con el negocio o el driver. |
| **Safe area** | Zona segura de pantalla en celulares con notch o barra inferior (especialmente iPhone). |
