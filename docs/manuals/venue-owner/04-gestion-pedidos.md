# Gestión de pedidos

## El KDS (Kitchen Display System)

El KDS es la pantalla principal donde gestionas los pedidos en tiempo real. Accede desde el menú lateral → **"Pedidos (KDS)"**.

El número en rojo sobre el ícono de KDS en el menú muestra cuántos pedidos están pendientes de atención.

---

## Estados de los pedidos

Los pedidos aparecen organizados por estado:

| Estado | Qué debes hacer |
|---|---|
| **Pendiente** | Un cliente hizo un pedido. Debes aceptarlo o rechazarlo. |
| **Aceptado** | Aceptaste el pedido. El cliente sabe que está en proceso. |
| **En preparación** | Marcaste que estás preparando el pedido. |
| **Listo** | El pedido está listo para ser recogido o entregado. |
| **Esperando driver** | El pedido está disponible para que un domiciliario lo tome. |
| **Completado** | El pedido fue entregado o recogido. |
| **Cancelado** | El pedido fue cancelado. |

---

## Flujo básico de un pedido

1. Llega un nuevo pedido → aparece en la columna "Pendiente" con un sonido de alerta.
2. Toca el pedido para ver el detalle (productos, cliente, método de entrega).
3. Toca **"Aceptar"** para confirmarlo, o **"Rechazar"** con un motivo.
4. Cuando empieces a preparar, toca **"En preparación"**.
5. Cuando esté listo, toca **"Listo"**.
6. Si es domicilio, el sistema asigna un driver automáticamente o un admin lo asigna manualmente.
7. Cuando el cliente recoge o el driver lo entrega, el pedido pasa a **"Completado"**.

---

## Historial de pedidos

En el menú lateral → **"Historial"**, puedes ver todos los pedidos pasados con filtros por fecha, estado y producto. Usa el historial para revisar el desempeño de días anteriores o resolver consultas de clientes.

---

## Chatear con un cliente

Dentro del detalle de un pedido, toca el ícono de mensaje para abrir el chat con ese cliente. Úsalo si necesitas aclarar algún detalle del pedido o informar un cambio.

---

## Notificaciones de nuevos pedidos

Asegúrate de tener las notificaciones activadas para recibir alertas inmediatas cuando llega un nuevo pedido. Sin notificaciones, podrías perder pedidos si no tienes la app abierta en ese momento.

Para verificar:
1. Ve a tu perfil → notificaciones.
2. Activa los permisos si no están activos.
