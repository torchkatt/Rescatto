# Gestión de productos

## Abrir el catálogo

En el menú lateral, toca **"Catálogo"**. Verás la lista de todos tus productos actuales con su estado, precio y stock.

---

## Crear un producto nuevo

1. En el catálogo, toca el botón **"+ Nuevo producto"**.
2. Completa el formulario:

| Campo | Descripción |
|---|---|
| **Nombre** | Nombre claro del producto (ej. "Almuerzo ejecutivo", "Paquete sorpresa de panadería") |
| **Descripción** | Qué incluye, ingredientes principales, porciones |
| **Precio original** | El precio habitual de ese producto |
| **Precio de rescate** | El precio especial para el cliente de Rescatto |
| **Stock** | Cuántas unidades tienes disponibles |
| **Fecha y hora límite** | Hasta cuándo se puede rescatar (ej. hoy a las 8:00 p.m.) |
| **Imagen** | Foto del producto. Usa imágenes claras y reales |
| **Categoría** | Tipo de producto (Desayuno, Almuerzo, Paquete sorpresa, etc.) |
| **Tipo** | Plato específico o paquete sorpresa |

3. Toca **"Crear producto"** para publicarlo.

### Precio de rescate recomendado

El precio de rescate suele ser entre el 30% y el 60% del precio original. Esto lo hace atractivo para el cliente y te permite recuperar algo del costo de producción.

---

## Editar un producto

1. En el catálogo, toca el producto que quieres modificar.
2. Toca el ícono de edición (lápiz).
3. Modifica los campos necesarios.
4. Guarda los cambios.

> Si el producto ya tiene pedidos activos, algunos campos no se pueden modificar hasta que esos pedidos se completen.

---

## Duplicar un producto

¿Tienes un producto similar al que ya creaste? Usa la función **"Duplicar"**:

1. En el catálogo, toca el ícono de copiar (al lado del lápiz) en el producto que quieres duplicar.
2. Se abre el formulario pre-lleno con los datos del original.
3. Modifica lo que necesites (nombre, precio, fecha).
4. La cantidad se reinicia a 0 y la fecha se establece en hoy para que la revises.
5. Guarda el nuevo producto.

---

## Gestión de stock

El stock es el número de unidades disponibles. Cuando llega a cero:
- El producto se muestra como "Agotado" en la app del cliente.
- No se pueden crear nuevos pedidos de ese producto.

**Alertas de stock bajo:** si el stock de un producto cae a 3 unidades o menos, verás una advertencia en el catálogo.

Para actualizar el stock:
1. Abre el producto.
2. Cambia el número en el campo "Stock".
3. Guarda.

---

## Precio dinámico

Puedes activar el precio dinámico en un producto para que baje automáticamente a medida que se acerca la hora límite:

- **Más de 4 horas:** precio original (sin descuento)
- **2–4 horas:** 10% de descuento
- **1–2 horas:** 20% de descuento
- **30–60 minutos:** 30% de descuento
- **Menos de 30 minutos:** 40% de descuento

El sistema actualiza los precios automáticamente cada 15 minutos.

Para activarlo, marca la casilla **"Precio dinámico"** al crear o editar el producto.
