# Rescatto | B2B Dashboard

**"Alta cocina, cero desperdicio."**

Rescatto es una plataforma SaaS que permite a hoteles, restaurantes y supermercados monetizar sus excedentes de comida de alta calidad mediante precios dinámicos y packs sorpresa.

Este repositorio contiene el **Dashboard Administrativo (MVP)** desarrollado en React + TypeScript + Vite.

## 🚀 Características Clave (MVP)

*   **Gestión de Inventario:** Creación de "Packs Sorpresa" y Platos Específicos.
*   **Dynamic Pricing Engine:** Simulación de algoritmo de precios basado en hora de cierre.
*   **KDS (Kitchen Display System):** Gestión de pedidos en tiempo real (Pendiente, Listo, Entregado).
*   **Analítica:** Dashboard de impacto financiero y ambiental (Kg salvados).

## 🛠️ Stack Tecnológico

*   **Frontend:** React 18, TypeScript, TailwindCSS.
*   **Build Tool:** Vite.
*   **Iconos:** Lucide React.
*   **Gráficos:** Recharts.
*   **Arquitectura:** Clean Architecture modular.

## 📦 Instalación Local

1.  Clonar repositorio.
2.  `npm install`
3.  `npm run dev`

## ☁️ Despliegue

Optimizado para despliegue continuo en **Vercel** o **Netlify**.
